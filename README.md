# NoHook

A simple Minecraft Fabric mod that allows you to customize the visibility of fishing bobbers and lines.

## Features

- Toggle visibility of fishing lines and bobbers.
- Option to only hide player-owned hooks.
- Option to hide lines only when an entity is hooked.
- Third-person camera support.
- Configurable through Mod Menu and Cloth Config.

## Requirements

- Minecraft 1.21.1
- Fabric Loader
- Fabric API
- Cloth Config API
